# JLPT N1 Question Formats

This document outlines the different types of questions that appear on the JLPT N1 test. Use this as your reference to detect the question type when user presents you with a multiple choice question without adding any specific instructions, in which case it is likely that the question comes from a past or mock JLPT exam.

---

## Section 1 言語知識・読解

### Question Type 1: 漢字読み

This question type tests the pronunciation of kanji. Pick the correct pronunciation of the word in brackets.

Example:

彼は今、新薬の研究開発に（挑んで）いる。
1．はげんで　2．のぞんで　3．からんで　4．いどんで

### Question Type 2: 文脈規定

This question type tests vocabulary definition understanding. Pick the most appropriate vocabulary to fill in the blank to complete the sentence.

Example:

私の主張は単なる（　）ではなく、確たる証拠に基づいている。
1．模索　2．思索　3．推測　4．推移

### Question Type 3: 言い換え類義

This question type tests vocabulary and synonyms. Pick the word that has the closest meaning to what is in brackets.

Example:

このマニュアルの説明は（ややこしい）。
1．明確だ　2．奇妙だ　3．複雑だ　4．簡潔だ 

### Question Type 4: 用法

This question type tests vocabulary usage. Pick the sentence that uses the word in question in the most appropriate way.

Example: 

いたわる
1．弱い立場の人を（いたわる）のは大切なことです。
2．山田さんはこれまでの努力を（いたわって）くれました。
3．母は孫が遊びに来たら、いつも（いたわって）いました。
4．政治家は国民の生活を（いたわる）べきです。

### Question Type 5: 文法形式の判断

This question type tests grammatical structures. Pick the most appropriate option to fill in the blank.

Example:

いまさら後悔してみた（　）、してしまったことは取り返しがつかない。
1．ところで　2．といえども　3．にせよ　4．ばかりに

### Question Type 6: 文の組み立て
This question type tests grammatical order. There are 4 consecutive blanks in the question sentence. Fill the blanks with all 4 options in the correct order. Provide the order as your answer.
Example:

人類は、生物的存在である ＿ ＿ ＿ ＿ 文化的存在でもある。

1．にもまして　2．他の　3．と同時に　4．どの種

### Question Type 7: 文章の文法

This question type tests grammatical usage and passage understanding. You will be provided a passage, in which there will be multiple numbered blanks. For each blank, pick the most appropriate option.

### Question Type 8: 内容理解（短文）

This question type tests reading comprehension. You will be given a short passage. Pick the most appropriate answer for the question asked.

### Question Type 9: 内容理解（中文）

This question type tests reading comprehension and is similar to Question Type 8, except the passage is of medium length, and there will be 2 questions that follow. Pick the most appropriate answers for the questions asked.

### Question Type 10: 内容理解（長文）

This question type tests reading comprehension and is similar to Question Type 8, except the passage is of extended length, and there will be 3 questions that follow. Pick the most appropriate answers for the questions asked.

### Question Type 11: 統合理解

This question type tests reading comprehension. You will be given two different reading materials on the same topic. Pick the most appropriate answers for the questions asked.

### Question Type 12: 主張理解（長文）
This question type tests reading comprehension and is similar to Question Type 10. Pick the most appropriate answers for the questions asked.

### Question Type 13: 情報検索

This question type tests reading comprehension and information gathering skills. You will be given a reading material not in typical paragraph format, such as menu, instructions, poster, data tables, etc. Pick the most appropriate answers for the questions asked.

## Section 2: 聴解

### Question Type 1: 課題理解
This question type tests listening comprehension. The question that follow is usually asking what one person is going to do next. Instead of the audio, you will be provided with the script of what was played. Pick the most appropriate answer for the question asked.

### Question Type 2: ポイント理解
This question type tests listening comprehension. The question that follow is usually asking for reasons or purposes of a certain action. Instead of the audio, you will be provided with the script of what was played. Pick the most appropriate answer for the question asked.

### Question Type 3: 概要理解
This question type tests listening comprehension. The question that follow is usually asking for the topic of discussion or what a certain person is trying to say. Instead of the audio, you will be provided with the script of what was played. Pick the most appropriate answer for the question asked.

### Question Type 4: 即時応答
This question type tests listening comprehension. It plays one sentence and then plays four potential responses to that sentence. Instead of the audio, you will be provided with the script of what was played. Pick the most appropriate response.

### Question Type 5: 統合理解
This question type tests listening comprehension. A long conversation is played. Pick the most appropriate answers for the questions asked.
